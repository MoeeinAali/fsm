    How to use

    • Add a state: double-click on the canvas
    • Add an arrow: shift-drag on the canvas
    • Move something: drag it around
    • Delete something: select it, press Delete or Cmd/Ctrl+⌫
    • Make accept state: double-click an existing state
    • Greek letter: backslash before the name (e.g. \beta)
    • Subscript: underscore before a digit (e.g. S_0)
    • Undo / Redo: Cmd/Ctrl+Z · Shift+Cmd/Ctrl+Z
